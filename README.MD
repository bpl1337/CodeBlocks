### Запуск проекта

### Простой способ
Откройте файл `index.html` в браузере, используя **Live Server** или аналогичный инструмент:

**В VS Code:**
1. Установите расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
2. Нажмите правой кнопкой мыши на `index.html`
3. Выберите **"Open with Live Server"**

Или просто используйте функцию **Live Preview**, если она доступна в вашем редакторе.
